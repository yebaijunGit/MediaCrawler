## 🚀 1. 快速启动
1. 进入文件夹 `H:\C#\tool\MediaCrawler\`。

## 🎛️ 2. JSON 配置面板 (推荐用法)
您无需修改任何 Python 核心代码，只需编辑根目录下的 **`user_settings.json`** 即可控制整个系统。

### 参数说明表：
| 参数名 | 功能描述 | 建议设置 |
| :--- | :--- | :--- |
| `SAVE_DATA_PATH` | **视频下载存到哪** | 保持默认或改为您的移动硬盘路径 |
| `BILI_QN` | **下载视频画质** | `120` (4K极其清晰), `80` (1080P) |
| `CRAWLER_MAX_NOTES_COUNT` | **单次自动抓取视频数** | `100` (如果博主视频多可以改大) |
| `PLATFORM` | **默认启动平台** | `bili` (B站), `xhs` (小红书) |
| `CRAWLER_TYPE` | **运行模式** | `creator` (博主主页搬运), `search` (关键词搜索) |

> **注意**：修改路径时请使用正斜杠 `/`，例如 `"G:/MyVideos"`。

## 🛠️ 5. 进阶：如何增加搬运目标？
如果您想增加要下载的博主，请修改：`config/bilibili_config.py`。
在 `BILI_CREATOR_ID_LIST` 的方括号 `[]` 内增加链接，用逗号隔开：
```python
BILI_CREATOR_ID_LIST = [
    "https://space.bilibili.com/1462935462",
    "https://space.bilibili.com/第二个博主ID",
]
```
